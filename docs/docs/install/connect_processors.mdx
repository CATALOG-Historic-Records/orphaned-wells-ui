# Connect trained processors to OGRRE

OGRRE relies on Google Document AI extractors for document processing. These must be trained before incorporating into OGRRE. To learn more about Document AI, check out their [documentation](https://cloud.google.com/document-ai#documentation). To connect your processor to OGRRE, follow this guide.
